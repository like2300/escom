Instructions pour l'importation

Convention de nommage
  • Format: Notes_Classe_Semestre_Année.xlsx
  • Exemple: Notes_L1-INFO_S1_2024.xlsx
  • Classe: Code de la classe (ex: L1-INFO)
  • Semestre: S1 ou S2
  • Année: Année académique (ex: 2024)
Structure du fichier Excel
  • Colonne A: Matricule (obligatoire)
  • Colonne B: Nom (obligatoire)
  • Colonne C: Prénom (obligatoire)
  • Colonne D: Note (0-20, obligatoire)
  • Colonne E: Type d'évaluation (Devoir/Session)
  • Pas de ligne d'en-tête supprimée
